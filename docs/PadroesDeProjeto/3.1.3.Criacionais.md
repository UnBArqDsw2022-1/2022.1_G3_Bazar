# GRASPs - Criacionais

## 1. Introdução
<p align="justify"> Determina qual classe deve ser responsável pela criação certos objetos. Atribua à classe B a responsabilidade de criar uma instância de A se pelo menos um desses for verdadeiro:
 - B contém ou agrega A, 
 - B registra a existência de A, 
 - B usa A,
 - B tem os dados necessários para a inicialização de A que serão passados ao construtor de A.
</p>

## 2. Metodologia
   <p align="justify"> Ao analisar o Diagrama de Classes produzido para o projeto, a analise sobre reponsbilidade de criação foi feita sobre cada classe a fim designar o papel criador:
   - A propria classe se cria,
   - A classe seria criada por outra classe.
   </p>
